# Screenshot Checklist

To ensure all visual assets are prepared for the Devpost submission and pitch deck, capture the following high-resolution screenshots:

## 1. Flutter Mobile App
- [ ] **Splash Screen**: The initial loading screen with the InfraGuard logo.
- [ ] **Secure State (Home)**: The dashboard showing green status and 'System Secure'.
- [ ] **Threat Detected (Alert)**: The red alert screen when a malicious payload is intercepted.
- [ ] **Notification**: A system push notification showing the threat alert on the mobile lock screen.
- [ ] **Action Panel**: The detailed payload view showing the **ALLOW**, **BLOCK**, and **QUARANTINE** buttons.
- [ ] **Settings Screen**: The configuration page showing Ngrok/WebSocket connection settings.

## 2. Web SOC Dashboard
- [ ] **Secure State (Dashboard)**: The web interface in a normal, secure state.
- [ ] **Threat Alert (Dashboard)**: The web interface flashing red during an intercepted attack.
- [ ] **Runtime Stream**: The live terminal/log view showing WebSocket events streaming in.
- [ ] **Incident Timeline**: The historical view of past events and admin decisions.
- [ ] **Payload Viewer**: Expanded JSON view of an intercepted malicious payload.

## 3. Architecture & Code
- [ ] **Architecture Pipeline**: A clean, exported image of the Mermaid System Architecture diagram.
- [ ] **FastAPI Terminal**: A screenshot of the backend terminal showing incoming JSON-RPC and WebSocket broadcasts.
- [ ] **AI Agent Terminal**: A screenshot of the simulated AI agent terminal firing payloads.
